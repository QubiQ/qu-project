======================
Modify Followers Issue
======================

This module extends the functionality of helpdesk ticket to allow you to modify the followers of a ticket.


Installation
============

You only have to install the module.


Configuration
=============

Is not necesary do any action. 


Usage
=====

To use this module, you need to:

#. Go to helpdesk.
#. Go to any ticket.
#. Click on the button in the header called *Modify Followers*.
#. Then you can add or delete the followers of the ticket.
